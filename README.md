# 🤖 ربات دریافت تصاویر محصولات دندانپزشکی

ربات خودکار برای دریافت تصاویر محصولات برندهای COXO، NSK و W&H از سایت coxotec.com.

## ✨ قابلیت‌ها

- جستجوی خودکار تصاویر محصولات
- پنل مدیریت وب برای تایید/رد تصاویر
- ذخیره تصاویر در پوشه `images/`
- تولید فایل `image_mapping.json`
- اجرای خودکار با GitHub Actions

## 🚀 نصب و اجرا

```bash
pip install -r requirements.txt
python scraper_bot.py
